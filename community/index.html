Join the PDI community

PDI is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use PDI? Find out more here:

  • Slack for users: Get support on Slack!
  • Gitlab for users: Get the code from Gitlab!

Develop and Contribute

If you want to get more involved by contributing to PDI, join us here:

  • GitLab: Contribute on Gitlab!
  • Slack: Discuss with developpers on Slack!
  • Jenkins: Check test results on Jenkins!
  • GitHub: Check repository mirrors on GitHub!
  • DockerHub: Try our Docker images from DockerHub!

You can find out how to contribute to these docs in our Contribution Guidelines.
